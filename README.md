# 🚀 Enhanced C++ Command-Line Calculator

A clean and user-friendly command-line calculator built with **C++**. This project demonstrates core programming concepts such as loops, switch-case logic, and conditional error handling.

## ✨ Key Features (New Version)
- **🔄 Continuous Execution:** Integrated a `while` loop, allowing you to perform multiple calculations without restarting the program.
- **🚪 Easy Exit:** Added an 'x' key option to safely exit the application.
- **🛠️ Smart Error Handling:** Includes a "Division by Zero" check to prevent program crashes.
- **🧹 Clean UI:** Improved console output formatting for better readability and a better user experience.

## 🛠️ Tech Stack
- **Language:** C++
- **Logic:** Switch-case statements & While loops
- **Compiler:** Compatible with Dev C++, VS Code, and all standard C++ compilers.

## 🚀 How to Run
1. **Download:** Copy the `Calculator.cpp` file to your computer.
2. **Compile:** Open the file in your IDE (e.g., Dev C++) and press **F11** to compile and run.
3. **Usage:** - Choose an operator: `+`, `-`, `*`, `/`
   - Enter two numbers.
   - Type `x` whenever you want to quit the program.
